# Eigenbijdrage.nl

Statische website voor eigenbijdrage.nl: een oplossing voor scholen, stichtingen en instellingen om ouderbijdragen, schoolkosten en andere bijdragen te innen met de software van Incassotool.net.

## Inhoud

- `index.html`: de volledige landingspagina
- `hoe-werkt-het.html`: uitleg van de betaalronde
- `voor-wie.html`: doelgroepen
- `prijzen.html`: pakketten en extra acties
- `contact.html`: demo-aanvraag
- `functies/*.html`: subpagina's per functionaliteit
- `styles.css`: styling en responsive layout
- `robots.txt`: crawler-toegang en sitemapverwijzing
- `sitemap.xml`: canonieke URL voor zoekmachines
- `llms.txt`: korte context voor LLMs en agents
- `llms-full.txt`: uitgebreide context voor LLMs en agents
- `humans.txt`: contact- en site-informatie
- `vercel.json`: Vercel headers en clean URLs

## Lokaal bekijken

Open `index.html` rechtstreeks in de browser.

## Canonieke URL

https://eigenbijdrage.nl/

## Deploy via Vercel

Dit project is een statische site. In Vercel kan de root van deze map worden gebruikt met:

- Framework preset: Other
- Build command: leeg laten
- Output directory: `.`
