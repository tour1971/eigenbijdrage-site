# Eigenbijdrage.nl

Statische website voor eigenbijdrage.nl: een oplossing voor scholen en instellingen om bijdragen te innen met de software van Incassotool.net.

## Lokaal bekijken

Open `index.html` rechtstreeks in de browser.

## Deploy via Vercel

Dit project is een statische site. In Vercel kan de root van deze map worden gebruikt met:

- Framework preset: Other
- Build command: leeg laten
- Output directory: `.`
